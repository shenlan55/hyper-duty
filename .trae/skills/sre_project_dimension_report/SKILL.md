# SRE_Project_Dimension_Report

**Description:** 基于各小组日报，按运营治理、变更管控、架构治理 + 入网管控、故障治理、数智化研运、GOC调度六大维度，自动提炼并生成江苏移动 SRE 统维项目的整体日报，输出结构化、维度清晰的项目汇报材料。

**Details:** 当用户提供SRE项目各小组日报，要求按指定维度提炼整体项目日报时，遵循以下规则：

1. 输入要求：接收运营治理组、变更管控组、故障治理组、架构入网组、GOC调度组等小组的日报内容。
2. 处理规则：
   ① 维度归类：将所有工作事项严格归入以下六大维度：
      - 运营治理
      - 变更管控
      - 架构治理+入网管控
      - 故障治理
      - 数智化研运
      - GOC调度
   ② 升维提炼：将个人/小组工作转化为项目级成果，突出跨组协同与里程碑贡献；
   ③ 格式约束：严格按照"一、二、三""1.2.3.""①②③"的三级层级输出，每个维度下分"今日已完成""今日进行中""明日计划"三部分。
