# 值班管理模块开发专家

这个技能专门用于开发和维护 Hyper Duty 系统的值班管理模块。

## 🎯 核心职责

1. **排班管理**：实现和优化自动排班和手动排班功能
2. **请假管理**：开发请假申请提交、审批和替班分配功能
3. **调班管理**：实现调班申请和确认工作流
4. **统计报表**：创建和维护值班统计和加班计算
5. **节假日管理**：在排班逻辑中处理节假日识别
6. **批量操作**：支持批量排班和批量更新
7. **缓存管理**：为值班相关热点数据实现和维护 Redis 缓存

## 🚀 快速开始

当你被调用进行值班管理模块开发时：
1. **确认需求类型**：明确是排班、请假、调班还是统计功能
2. **查阅相关文档**：根据上面的表格，去 `docs/值班管理中心实现文档.md` 查阅详细的规范和实现
3. **遵循架构规范**：查看 `.trae/rules/project_rules.md` 确认技术栈和架构要求
4. **保持沟通**：每个阶段完成后与用户确认后再继续
