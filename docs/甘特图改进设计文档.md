3. **高度优化** - 完全移除高度限制，自动撑开
4. **用户体验优化** - 拖拽调整、详情弹窗等

### 1.2 改进目标
- 提升甘特图的可读性和交互性
- 支持灵活的时间范围筛选
- 优化父子任务的层级展示
- 提供便捷的任务管理功能

**预设选项**：

2. 选择"自定义"时，显示两个日期选择器
3. 时间范围变化时，重新加载任务数据和时间轴

#### 2.1.2 视图切换器
**功能描述**：
- 切换时间轴的显示粒度

**选项**：
- 日视图：每天一个单元格
- 周视图：每周一个单元格
- 月视图：每月一个单元格

**交互逻辑**：
- 默认使用日视图
- 切换视图时，重新计算时间轴和任务条位置

#### 2.1.3 展开/折叠全部
**功能描述**：
- 一键展开或折叠所有任务的子任务

**交互逻辑**：
- "展开全部"：将所有任务的 `collapsed` 状态设为 `false`
- "折叠全部"：将所有任务的 `collapsed` 状态设为 `true`

### 2.2 左侧任务列表

#### 2.2.1 树形结构展示
**展示风格**：方案C混合风格

**视觉效果**：
```
▶ 父任务1
├ ▶ 子任务1-1
┃ └ 子任务1-1-1
└ 子任务1-2
▶ 父任务2
```

**实现要点**：
- 使用CSS伪元素 `::before` 和 `::after` 绘制竖线
- 折叠/展开箭头使用Element Plus的Icon组件
- 根据 `taskLevel` 计算缩进距离
- 每个任务的 `collapsed` 状态独立管理

#### 2.2.2 任务状态标识
**状态颜色**：
- 🟢 进行中：绿色
- 🔵 已完成：蓝色
- ⚪ 未开始：灰色
- 🟡 已暂停：黄色

**实现方式**：
- 在任务名称前显示状态图标
- 使用CSS类控制图标和文字颜色

#### 2.2.3 置顶标识
**视觉效果**：
- 置顶任务用红色边框标识
- 或在任务名称前显示 📌 图标

### 2.3 右侧时间轴

#### 2.3.1 时间轴刻度
**日视图**：
- 每个单元格代表一天
- 显示格式："6/2"

**周视图**：
- 每个单元格代表一周
- 显示格式："第23周"

**月视图**：
- 每个单元格代表一个月
- 显示格式："2026年6月"

#### 2.3.2 特殊日期标识
- **今日高亮**：蓝色背景 `#ecf5ff`
- **周末标识**：浅灰色背景 `#fafafa`

#### 2.3.3 任务条
**视觉样式**：
- 圆角矩形 `border-radius: 4px`
- 渐变背景色（根据进度）
- 中间显示进度百分比文字

**颜色规则**：
- 已完成（status=3）：`#67c23a`（纯绿色）
- 已暂停（status=4）：`#909399`（纯灰色）
- 进行中（progress>=60）：绿色渐变
- 进行中（progress>=30）：橙色渐变
- 进行中（progress<30）：红色渐变

**交互功能**：
1. **拖拽移动**：
   - 拖拽任务条中间部分，可移动整个任务的时间范围
   - 释放时更新任务的 `startDate` 和 `endDate`

2. **拖拽调整长度**：
   - 拖拽任务条左端，调整开始日期
   - 拖拽任务条右端，调整结束日期

3. **点击查看详情**：
   - 点击任务条或任务名称，打开任务详情弹窗
   - 弹窗显示任务完整信息和操作按钮

### 2.4 同步滚动
**功能描述**：
- 左侧任务列表和右侧时间轴保持垂直滚动同步

**实现方式**：
- 监听右侧滚动容器的 `scroll` 事件
- 同步更新左侧容器的 `scrollTop`

### 3.2 核心函数

#### 3.2.1 构建树形结构
```javascript
/**
 * 将扁平任务列表转换为树形结构
 * @param {Array} tasks 扁平任务列表
 * @returns {Array} 树形结构的任务列表
 */
const buildTaskTree = (tasks) => {
  const taskMap = new Map()
  const roots = []
  
  // 先创建映射
  tasks.forEach(task => {
    taskMap.set(task.id, { ...task, children: [] })
  })
  
  // 构建树
  tasks.forEach(task => {
    const node = taskMap.get(task.id)
    if (task.parentId && taskMap.has(task.parentId)) {
      taskMap.get(task.parentId).children.push(node)
    } else {
      roots.push(node)
    }
  })
  
  return roots
}
```

#### 3.2.2 计算时间范围
```javascript
/**
 * 根据预设选项计算时间范围
 * @param {String} range 时间范围选项
 * @returns {Object} { startDate, endDate }
 */
const calculateTimeRange = (range) => {
  const now = new Date()
  let start, end
  
  switch (range) {
    case 'week':
      // 本周：从周一到周日
      const day = now.getDay() || 7
      start = new Date(now)
      start.setDate(now.getDate() - day + 1)
      end = new Date(start)
      end.setDate(start.getDate() + 6)
      break
      
    case 'month':
      // 本月
      start = new Date(now.getFullYear(), now.getMonth(), 1)
      end = new Date(now.getFullYear(), now.getMonth() + 1, 0)
      break
      
    case 'quarter':
      // 本季度
      const quarter = Math.floor(now.getMonth() / 3)
      start = new Date(now.getFullYear(), quarter * 3, 1)
      end = new Date(now.getFullYear(), quarter * 3 + 3, 0)
      break
      
    case 'year':
      // 本年
      start = new Date(now.getFullYear(), 0, 1)
      end = new Date(now.getFullYear(), 11, 31)
      break
      
    default:
      // 默认：任务的最小和最大日期，前后各扩展7天
      return calculateDefaultRange()
  }
  
  return {
    startDate: formatDate(start),
    endDate: formatDate(end)
  }
}
```

#### 3.2.3 生成时间轴
```javascript
/**
 * 生成时间轴日期列表
 * @param {String} startDate 开始日期
 * @param {String} endDate 结束日期
 * @param {String} viewMode 视图模式
 * @returns {Array} 日期列表
 */
const generateDateList = (startDate, endDate, viewMode) => {
  const dates = []
  const current = new Date(startDate)
  const end = new Date(endDate)
  
  while (current <= end) {
    dates.push(formatDate(current))
    
    switch (viewMode) {
      case 'day':
        current.setDate(current.getDate() + 1)
        break
      case 'week':
        current.setDate(current.getDate() + 7)
        break
      case 'month':
        current.setMonth(current.getMonth() + 1)
        break
    }
  }
  
  return dates
}
```

#### 3.2.4 计算任务条位置
```javascript
/**
 * 计算任务条的位置和宽度
 * @param {Object} task 任务对象
 * @param {Array} dateList 日期列表
 * @param {String} viewMode 视图模式
 * @returns {Object} { left, width }
 */
const calculateTaskBarStyle = (task, dateList, viewMode) => {
  if (!task.startDate || !task.endDate || dateList.length === 0) {
    return {}
  }
  
  const startIndex = dateList.indexOf(task.startDate)
  const endIndex = dateList.indexOf(task.endDate)
  
  if (startIndex === -1 || endIndex === -1) {
    return {}
  }
  
  const cellWidth = viewMode === 'day' ? 40 : viewMode === 'week' ? 80 : 120
  
  return {
    left: startIndex * cellWidth + 'px',
    width: (endIndex - startIndex + 1) * cellWidth + 'px'
  }
}
```

## 7. v1.1 BugFix 技术细节

### 7.1 折叠状态管理（修复展开/折叠全部无效）

**问题**：`flatTaskList` 是 computed，每次重新计算时 `buildTaskTree` 都生成新的 `collapsed: false` 对象，导致在 computed 结果上修改 `collapsed` 属性在下一次计算时丢失。

**修复**：用独立的 `ref(new Set())` 存储已折叠的任务 ID。

```javascript
const collapsedTaskIds = ref(new Set())

const toggleCollapse = (taskId) => {
  const newSet = new Set(collapsedTaskIds.value)
  newSet.has(taskId) ? newSet.delete(taskId) : newSet.add(taskId)
  collapsedTaskIds.value = newSet  // 创建新实例触发响应式
}

const expandAll = () => { collapsedTaskIds.value = new Set() }
const collapseAll = () => {
  // 收集所有有子任务的ID
  collapsedTaskIds.value = new Set(flatTaskList.value.filter(t => hasChildren(t.id)).map(t => t.id))
}
```

### 7.2 时间条边界夹紧（修复切时间范围后时间条消失）

**问题**：`getBarStyle` 中 `indexOf` 找不到目标任务日期时直接返回 `{}`，导致任务跨时间范围边界时时间条消失。

**修复**：提前判断完全超出范围的情况，部分超出则夹紧到首/尾。

```javascript
const getBarStyle = (task) => {
  const taskStart = normalizeDateStr(task.startDate)
  const taskEnd = normalizeDateStr(task.endDate)
  
  // 完全在可视范围外 → 不显示
  if (taskEnd < firstVisible || taskStart > lastVisible) return {}
  
  // 部分超出 → 夹紧
  if (startIndex < 0) startIndex = 0
  if (endIndex < 0) endIndex = dateList.value.length - 1
}
```

### 7.3 表头横向滚动同步（修复横向滚动时日期标签不跟随）

**问题**：表头在 `.gantt-body` 滚动区域外，横向滚动时表头不动导致日期与任务条错位。

**修复**：表头设 `overflow: hidden` + JS 同步 `scrollLeft`。

```javascript
const handleScroll = (e) => {
  fixedBodyRef.value.scrollTop = e.target.scrollTop        // 竖向同步
  timelineHeaderRef.value.scrollLeft = e.target.scrollLeft  // 横向同步
}
```

### 7.4 时间范围任务裁剪（修复不在范围内的任务也显示）

**问题**：所有任务无条件显示，即使完全不在当前选中的时间范围内。

**修复**：`pruneTree` 递归裁剪，`isTaskInRange` 做日期交集判断。

```
保留规则：isTaskInRange(node) === true OR node.children.length > 0
即：自身在范围内，或作为有范围内子任务的父节点来维持层级。
```

```javascript
const pruneTree = (nodes) => {
  return nodes.filter(node => {
    if (node.children?.length) node.children = pruneTree(node.children)
    return isTaskInRange(node) || (node.children && node.children.length > 0)
  })
}
```

`flatTaskList` 和 `visibleFlatList` 均改为从 `pruneTree(buildTaskTree(...))` 构建。

### 7.5 列对齐与表头固定

| 问题 | 修复 |
|------|------|
| 左右行高不一致 | 统一 `height: 41px + border-box` |
| 浏览器页面级滚动 | 容器 `height: calc(100vh - 260px)` 迫使内部滚动 |
| 表头不固定 | 滚动从 `col` 移到 `body`，表头天然固定 |
| 下拉框宽度跳变 | 固定 `style="width: XXXpx"` |
| 日周月按钮换行 | `flex-wrap: nowrap; white-space: nowrap` |
